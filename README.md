## 🔍 Project Overview

This project presents an interactive **Power BI dashboard** built using Salesforce CRM data. It provides insights into **lead generation, conversion rates, and sales opportunities**, helping businesses make data-driven decisions.

The dashboard is divided into:
- 📌 Lead Dashboard  
- 📌 Opportunity Dashboard  

---

## 📌 Lead Dashboard Insights

### ✅ Key Metrics
- **Total Leads:** 10K  
- **Converted Opportunities:** 411  
- **Conversion Rate:** 10.33%  
- **Converted Accounts:** 1016  
- **Expected Revenue:** 36.62M  

### 📊 Analysis
- **Lead by Stage:** Distribution across stages like Nurturing, Prospect, MQL, SQL, Converted, etc.  
- **Lead by Source:** Performance of channels such as Website, Inside Sales, Trade Shows, Webinars  
- **Lead by Month:** Monthly trends and seasonal patterns  
- **Lead by Industry:** Top contributing industries like Safety & Security and Life Sciences  

---

## 📌 Opportunity Dashboard Insights

### ✅ Key Metrics
- **Total Forecast:** 184.14M  
- **Conversion Rate:** 31.06%  
- **Win Rate:** 42.77%  
- **Loss Rate:** 57.23%  
- **Total Opportunities:** 4646  

### 📊 Analysis
- **Revenue Forecast by Category:** Closed, Pipeline, Best Case, Forecast, Omitted  
- **Opportunities by Year:** Year-wise growth trend  
- **Closed vs Open Opportunities:** Deal status distribution  
- **Won Opportunities:** Comparison of won vs lost deals  
- **Top Industries:** Biopharma, Federal, Energy, and more
